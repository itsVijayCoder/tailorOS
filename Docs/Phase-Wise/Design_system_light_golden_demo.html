Motion and accessibility tokens

Animate only meaningful feedback: button press, drawer open, command palette, status change, new WhatsApp message, and route content transition. Prefer transform and opacity. Respect prefers-reduced-motion.

--motion-fast: 120ms; --motion-medium: 220ms; --ease-premium: cubic-bezier(.2,.8,.2,1); @media (prefers-reduced-motion: reduce) { *, *::before, *::after { transition-duration: .01ms !important; } }

Design checklist

  • Use light golden surfaces for warmth and premium local-business identity.
  • Use dark ink text for readability on mobile.
  • Keep command search accessible from every page with Ctrl + K.
  • Status colours must communicate action: green ready/success, amber pending/warning, red blocked/failure, teal WhatsApp/service.
  • Every component has loading, empty, error, disabled, success, and permission-denied states.
